# Prova técnica API

#Pré-requisitos
Para que seja possivel rodar essa aplicação é necessário:
-Java 1.8.0
-Maven 3.8.3
-Junit 4.13.2
-Rest Assured 3.3.0 

#Inicializando
- Ao abrir o projeto de um refesh no maven ou instale as dependências com o seguinte comando no terminal "mvn clean install";
- Abra a pasta src;
- Abra a pasta java;
